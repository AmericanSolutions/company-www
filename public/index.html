THERMS - Webapp Business Management Portal

THERMS (The Report Management System) started as an internal business tool/project for a client back in 2012 (CA Security Agency). The primary idea of the webapp was to create a web service application where the business employees could login to record information about their shift/job duties, scheduling, and internal messaging. The webapp evolved and rapidly gained popularity!

In 2016 the client requested a complete rewrite of the project which was originally in PHP. This project was converted using a Full-Stack solution with Node.js & React.js. The webapp is now modular, extensible, and serving over 10,000 users daily! Our team has implemented features such as video/image upload, custom template emailing, scheduling, mobile application platform, network security features such as 2 factor authentication, and so much more.

The experiences from this project continue to come in the door as the American Software team performs maintenance and feature requests on the app. This project has become a benchmark for our team to discuss and compare other clients projects and feature on when discussing performance, UI/UX, and raw code execution for complex features.

Client: www.THERMS.io
Date: 2016/2017
Service: Full-Stack, Web App, Mobile App, UX, Big-Data Database Hosting
